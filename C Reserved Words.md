# C Reserved Words

The set of reserved words below is used to build up the basic instructions of C; you can not use them in programs you write.

- auto
- break
- case
- char
- continue
- default
- do
- double
- else
- entry
- extern
- float 
- for 
- goto
- if
- int
- long
- register
- return
- short 
- sizeof
- static
- struct
- switch
- typedef
- union
- unsigned
- while

# In modern implementations:

- const
- enum 
- signed 
- void
- volatile
