# Elixir Reserved Words

The following keywords are reserved for use by Elixir. Do not use any of these words as attribute names in expressions.

- A
  - after
  - and
- C
  - catch
- D
  - do
- E
  - else
  - end
- F
  - false
  - fn  
- I
  - in
  - inbits
  - inlist
- N
  - nil
  - not
- O
  - or
- R
  - rescue  
- T 
  - true  
- W
  - when
- X   
  - xor
